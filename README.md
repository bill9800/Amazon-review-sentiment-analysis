# Amazon-review-sentiment-analysis

**Preprocessing**: TFIDF-Vectorizer

**Selected Models**: Decision Tree, Logistic Regression, Neural Network(10 * 10), Random Forest, Ensemble(hard voting: LR+NN+RF)
